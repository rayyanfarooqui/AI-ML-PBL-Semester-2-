# Phase 2 - AI/ML PBL Project

This folder contains the Phase 2 work of our AI/ML PBL project on **Automated Credit Risk Categorization for Loan Approval**.

In this phase, we focused on model development and training. We applied multiple machine learning algorithms including **Logistic Regression, K-Nearest Neighbors (KNN), and Support Vector Machine (SVM)** to classify credit risk.

We trained and tested the models using the prepared dataset and compared their performance based on evaluation metrics such as Accuracy, Precision, Recall, and F1 Score.

This phase helps in identifying the most suitable model for accurate credit risk prediction and prepares the system for final optimization in the next phase.
