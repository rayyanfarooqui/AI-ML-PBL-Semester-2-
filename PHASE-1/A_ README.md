# Phase 1 - AI/ML PBL Project

This folder contains the Phase 1 work of our AI/ML PBL project on **Automated Credit Risk Categorization for Loan Approval**.

In this phase, we focused on understanding the problem statement and planning the project approach. We identified key features such as Credit Score, Annual Income, and Debt-to-Income Ratio, and designed the initial machine learning pipeline.

We planned to use **Logistic Regression** for classification and defined evaluation metrics such as Precision, Recall, and F1 Score.

This phase forms the foundation for model development and implementation in the upcoming phases.
