# Phase 3 - AI/ML PBL Project

This folder contains the Phase 3 work of our AI/ML PBL project on **Automated Credit Risk Categorization for Loan Approval**.

In this phase, we implemented the final prediction system by taking **user input** for credit risk evaluation. The trained models including **Logistic Regression, K-Nearest Neighbors (KNN), and Support Vector Machine (SVM)** were used to make predictions based on user-provided data.

We tested different model inputs and observed their outputs to determine the most effective approach for real-time prediction.

This phase completes the project workflow by enabling interactive credit risk classification using machine learning models.
